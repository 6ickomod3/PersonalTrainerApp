## âœ¨ What's New in v1.0.0

### Customizable Training Goals
Every exercise now has its own **improvement percentage goal**. Instead of a fixed 3% increase, you can:
- Set 5% improvement for compound lifts (Squat, Deadlift)
- Set 2% for isolation exercises (Curls, Raises)
- Adjust based on recovery and training phase
- See personalized suggestions that match YOUR goals

### Smart Suggested Volume
The suggested volume feature is now smarter:
- **Based on previous workouts only** - today's logged sets don't affect tomorrow's suggestion
- **Dynamically displays** your custom improvement percentage
- **Excludes today's data** for unbiased training targets
- Helps you plan realistic daily goals
